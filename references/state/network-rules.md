# 联网搜索规则

## 🌐 联网搜索规则

允许联网，这是翻译质量的重要保障：

### 必须联网的情况
1. **专业术语查证**：不确定的科技术语、医学名词、法律概念 → 搜索权威来源验证
2. **新兴词汇查证**：网络新词、行业新术语、缩写 → 搜索最新用法和译法
3. **文化典故查证**：不熟悉的典故、人物、事件 → 搜索了解背景后再翻译
4. **翻译规范查询**：特定领域的翻译标准、术语表 → 搜索行业翻译规范
5. **货币与汇率**：实时汇率 → 搜索最新数据
6. **多译法对比**：同一术语有多种译法时 → 搜索使用频率和权威来源，选最优

### 搜索原则
1. **优先权威来源**：ISO标准、国际组织术语库（UNTERM/IATE）、国家级术语标准、学术数据库
2. **交叉验证**：重要术语搜≥2个来源确认
3. **时效性优先**：优先最新结果，旧译法可能已被淘汰
4. **领域区分**：同一个词在医学、法律、工程中可能有不同译法，搜索时指定领域
5. **不依赖机器翻译**：Google Translate/DeepL的结果只能参考，不能作为权威依据
6. **诚实标注来源**：引用查到的译法时注明出处

### 搜索日志
对于专业翻译，记录搜索过程：
```
[搜索]
术语：photosynthesis
领域：生物学
来源1：全国科学技术名词审定委员会 → 光合作用
来源2：Wikipedia 中文版 → 光合作用
结论：采用"光合作用"，有权威术语标准支撑
```
