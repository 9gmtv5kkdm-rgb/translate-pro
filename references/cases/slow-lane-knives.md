# 第七层：翻译慢车道——四把诊断刀（细则）

## 🔬 第七层：翻译慢车道——四把诊断刀

> **定位**：快车道案例库没覆盖的翻译问题，用诊断刀深度追问。不是扫描匹配——是逐句感知→定格追问→找到根→修正→回翻验证。

### 执行顺序

```
快车道先跑（案例筛查秒改）
  │
  ▼
四把诊断刀逐句追问
  ├─ 🔪第一把：准确吗？
  ├─ 🔪第二把：通顺吗？
  ├─ 🔪第三把：地道吗？
  └─ 🔪第四把：传神吗？
  │
  ▼
冲突裁决 → 快慢冲突以慢为准
```

---

#### 🔪 第一把刀：信息准确吗？——漏译/增译/误译检查

> **译文的信息保真度是否100%？没多没少没歪？**

**追问清单**：
- □ 有没有漏掉的句子、短语、关键词？（尤其注意否定词not/never、程度词always/only）
- □ 有没有原文没有但译文擅自添加的内容？（"译者觉得需要补一句"=增译）
- □ 数字、日期、金额、百分比——逐个核对过吗？
- □ 情态动词的力度是否精确对应？（must=必须、shall=应/须、may=可、will=将）
- □ 同一术语在全文是否使用了统一译名？
- □ 专有名词（人名、地名、公司名、产品名）的译法是否正确/一致？

**准确度标尺**：
| 级别 | 表现 | 操作 |
|------|------|------|
| ⭐⭐⭐⭐⭐ 零失误 | 所有信息点核对无误 | 直接通过 |
| ⭐⭐⭐⭐ 微瑕 | 个别非关键修饰词有偏差但不影响理解 | 微调 |
| ⭐⭐⭐ 有遗漏 | 漏掉1-2个信息点但非核心 | 补译+标注 |
| ⭐⭐ 关键失误 | 数字/日期/否定词/法律术语有误 | 重译该段 |
| ⭐ 严重歪曲 | 核心意思翻反了 | 全文重译 |

**修正路径**：
1. 回原文逐句对齐——原文一句，译文一句，勾掉已翻译的内容
2. 特别检查否定词（not/never/none）、数量词（all/most/some/few）的翻译
3. 法律/合同文本额外检查shall/must/may的力度对应

---

#### 🔪 第二把刀：表达通顺吗？——翻译腔检测

> **译文读起来像目标语原创吗？还是有"翻译味"？**

**追问清单**：
- □ 大声读出译文——有卡住的地方吗？有需要回头看才能理解的地方吗？
- □ 有没有一眼能看出是从源语直译过来的句式结构？
- □ 被动语态在目标语中自然吗？（中文少用被动，英文被动正常）
- □ 长句是否需要切分？（中文建议单句≤40字，英文单句≤30词）
- □ 有没有连续出现同样的句式结构（"通过……""基于……""对于……"）？
- □ 代词指代清晰吗？（"它""这"指什么？不同语言的代词使用频率差异很大）

**通顺度标尺**：
| 级别 | 表现 | 操作 |
|------|------|------|
| ⭐⭐⭐⭐⭐ 完全自然 | 看不出是翻译的 | 直接通过 |
| ⭐⭐⭐⭐ 基本流畅 | 1-2处轻微生硬但不影响阅读 | 微调 |
| ⭐⭐⭐ 明显翻译腔 | 多处"被字句""的地得"堆积、欧化长句 | 重构句式 |
| ⭐⭐ 读起来费劲 | 需要读者自己在脑中"翻译回原文"才能理解 | 大幅改写 |
| ⭐ 完全读不懂 | 逐词硬译，不知所云 | 重译 |

**修正路径**：
1. 遮住原文，只读译文——如果能自然理解 → 通过
2. 识别翻译腔高频词（"在……的情况下""对于……""起到了……的作用""被……所……"）→ 删除或重构
3. 长句切分：找从句边界，中文用逗号/分号断句，英文控制句长

---

#### 🔪 第三把刀：场景地道吗？——文化适配检查

> **在这个目标场景下，目标读者会这样说/写吗？**

**追问清单**：
- □ 称呼和敬语符合目标文化的场景惯例吗？
- □ 如果这是商务邮件——抬头、结尾、语气是否符合目标语言的商务邮件格式？
- □ 如果这是社交媒体——用了网络用语吗？语气够松弛吗？
- □ 如果这是演讲——排比的气势保留了吗？金句的节奏还在吗？
- □ 文化元素处理了吗——典故归化/注释/替换？
- □ 有没有文化禁忌需要避开？（颜色/数字/宗教表达/性别称呼）
- □ 日期格式、计量单位、货币——按目标市场习惯转换了吗？

**地道度标尺**：
| 级别 | 表现 | 操作 |
|------|------|------|
| ⭐⭐⭐⭐⭐ 完美融入 | 目标读者不会察觉这是翻译 | 直接通过 |
| ⭐⭐⭐⭐ 基本合适 | 1-2处称呼/格式微瑕 | 微调 |
| ⭐⭐⭐ 有些违和 | 称呼或格式不符合场景惯例 | 调整场景适配 |
| ⭐⭐ 明显不适 | 文化元素未经处理，直译导致误解或冒犯 | 文化适配重做 |
| ⭐ 严重失礼 | "美女"直译成"beautiful"等社交灾难 | 紧急修正 |

**修正路径**：
1. 用场景路由S1-S8的"场景红线"逐条对照——有没有触碰红线？
2. 称呼/结尾/格式按目标场景的惯例重写（不保留源语的客套形式）
3. 文化典故三级处理：可归化→等效替换；需保留→直译+注释；无足轻重→意思替代

---

#### 🔪 第四把刀：风格传神吗？——语气/情感/节奏匹配

> **原文的语气、情感、节奏——译文抓住了吗？还是翻"平"了？**

**追问清单**：
- □ 原文是幽默的——译文也幽默吗？（不是"翻译了笑话的内容"，是"译文让人笑了"）
- □ 原文是严肃的——译文也严肃吗？（没有因为"意译"而泄了气）
- □ 原文有排比/重复的修辞——译文保留了这个修辞效果吗？
- □ 原文的情感温度——热情/冷静/温暖/讽刺——译文一致吗？
- □ 原文的句式节奏——长短句交替、断句的呼吸感——译文重建了吗？
- □ 如果是文学/演讲/广告文案——文采够吗？（金句不能翻成白开水）

**风格度标尺**：
| 级别 | 表现 | 操作 |
|------|------|------|
| ⭐⭐⭐⭐⭐ 完美还原 | 从译文能感受到和原文一样的"情绪温度" | 直接通过 |
| ⭐⭐⭐⭐ 大方向对 | 风格基调正确，个别修辞效果有差 | 微调修辞 |
| ⭐⭐⭐ 风格偏差 | 正式度或情感浓度与原文偏离明显 | 调整语气层级 |
| ⭐⭐ 风格丢失 | 幽默翻干了、排比翻断了、金句翻平了 | 重构该段 |
| ⭐ 风格全错 | 讽刺翻成了赞美，或反之 | 重译 |

**修正路径**：
1. 对比原文和译文的"情感曲线"——哪里原文有起伏但译文平了？
2. 修辞手法对等转换：排比→排比、比喻→找目标语中功能对等的比喻、双关→等效幽默/标注
3. 大声朗读原文和译文各一遍——感受节奏是否匹配

---

### 诊断刀协作规则

| 规则 | 说明 |
|------|------|
| **按序追问** | 先准确→再通顺→再地道→再传神。不可跳跃——L1没过，追L4无意义 |
| **快车道优先** | 案例库里已经有模式的，直接修正，不进入诊断刀。除非当前场景特殊需要重新判断 |
| **冲突以慢车道为准** | 案例库说"改"，诊断刀判断在当前场景合理 → 保留 |
| **定格追问** | 感知到"不对劲"时，不凭直觉改。先定位到是哪一把刀的哪一条追问，再修正 |
| **回翻验证** | 修正后大声读一遍译文全文——还有没有新的不对劲？→ 有就再走一轮 |
