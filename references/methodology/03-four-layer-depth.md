# 第三层：四层翻译深度（细则）

## 🎯 第三层：四层翻译深度

每一层是递进关系，不是并列关系。先保证L1，再追求L2，依此类推。

### L1：信息准确——"不说错"

> 这是翻译的底线。L1出问题，后面的都白做。

**核心要求**：
- 不增（原文没有的，不擅自添加）
- 不漏（原文有的，一个不落）
- 不改（原文的意思，不擅自扭曲）
- 术语一致（同一术语全篇统一译名）
- 数据精准（数字、日期、金额、百分比零误差）

**常见L1错误**：
- 把"billion"翻成"亿"（应为"十亿"）——数字级错误
- 把"next Friday"翻成"下周五"（如果今天是周四，next Friday是明天还是下周？）——时间歧义
- 把"may"翻成"必须"（may是许可/可能，不是义务）——法律后果
- 漏译一个"not"——意思全反

**L1自检清单**：
□ 所有数字/日期/金额/百分比核对过了吗？
□ 否定词/程度词/可能性词（not/never/always/may/must）翻译准确吗？
□ 专业术语的译名是否一致（前后有无交替用不同译名）？
□ 有没有漏掉的句子、短语？

### L2：表达通顺——"像人话"

> L1保证不错，L2保证不别扭。

**核心要求**：
- 摆脱源语结构束缚——用目标语的自然语序表达
- 消除翻译腔——不是"把每个词换成目标语"，是"用目标语重新说一遍"
- 长句合理切分——中文不超过40字/句，英文不超过30词/句
- 语序符合目标语习惯——中文先因后果、英文可先果后因

**常见翻译腔（中译方向）**：

| 翻译腔 | 自然表达 |
|--------|----------|
| 在……的情况下 | 删除，直接说条件 |
| 对于……进行了…… | 直接说"做了……" |
| 起到了……的作用 | 直接说"作用是……"或删除 |
| 基于……的考虑 | "因为……" |
| 被广泛认为 | "公认""普遍认为"（中文少用被动） |
| 得到了……的结果 | "结果是……" |
| …之一 | 如非必要则删除 |
| 当我们……的时候 | 直接说"……时" |

**L2自检清单**：
□ 大声读出来顺口吗？有没有地方卡住？
□ 有没有一眼能看出是从X语直译过来的结构？
□ 长句需要切分吗？
□ 被动语态在目标语中自然吗？

### L3：场景地道——"符合目标文化习惯"

> L2保证通顺，L3保证这个场景就该这么说。

**核心要求**：
- 文化的归文化，语言的归语言——文化元素需要转换，不只是翻译
- 目标读者读到不会有"这是翻译的"感觉
- 符合目标语言的场景惯例（邮件格式、称呼习惯、敬语体系等）

**场景地道化操作要点**：

#### 称呼系统转换
| 源语 | 目标语示例 | 说明 |
|------|-----------|------|
| Dear Mr. Smith | 史密斯先生/尊敬的史密斯先生 | 英文Dear ≠ 中文"亲爱的" |
| To Whom It May Concern | 敬启者/相关负责人 | 中文有专门的公文抬头 |
| Hey guys | 各位/大家 | 性别的中性化处理 |
| 王总/李局 | Mr. Wang/Mr. Li（或根据场景保留头衔） | 中文头衔体系在英文中需转换 |

#### 敬语体系处理
- 日语 → 中文：敬语（です・ます体）通常转化为礼貌用词，不逐词对应
- 中文 → 日语：根据关系亲疏确定です・ます还是简体
- 韩语 → 中文：합쇼체（最敬阶）→ 正式敬语；해요체 → 礼貌用语；해체 → 日常口语
- 英文 → 中文：please不总是"请"，大量would/could需要根据上下文判断语气强弱

#### 文化典故处理
- **可归化的**：中文"画蛇添足"→ 英文可说"gilding the lily"（找到功能对等的成语）
- **需保留+注释的**：中国特色的"大跃进""双十一"→ 音译/直译 + 脚注解释
- **需替换的**：如果典故在目标语中毫无认知度且非核心信息 → 用意思替代

**L3自检清单**：
□ 目标语的读者会这样说吗？在这个场合下？
□ 称呼和敬语符合目标文化习惯吗？
□ 文化元素处理了吗（归化/注释/替换）？
□ 格式符合目标语言场景的默认格式吗？

### L4：风格匹配——"保留原文风格和语气"

> L3保证地道，L4保证这个"人"就是这个味儿。

**核心要求**：
- 传达原文的语气——是讽刺、是幽默、是严肃、是温情？
- 保留原文的节奏——长短句交替、排比、重复等修辞效果
- 还原原文的声调——正式/随意、权威/谦逊、热情/冷静

**风格维度适配**：

| 风格维度 | 识别方法 | 翻译原则 |
|----------|----------|----------|
| **正式度** | 用词（furthermore vs also）、句式复杂度 | 保持对应的正式层级 |
| **情感浓度** | 感叹号、情感词、修辞手法 | 找到对应情感强度的表达 |
| **节奏感** | 句长分布、排比、重复 | 在目标语中重建相似的节奏模式 |
| **幽默类型** | 文字游戏/双关/讽刺/夸张 | 文字游戏优先找对等→找不到就换同效果幽默 |
| **身份标签** | 方言、阶层用语、年龄特征 | 用目标语中社会功能对等的表达 |

**L4自检清单**：
□ 原文是幽默的，译文也幽默吗？
□ 原文是严肃的，译文也严肃吗？
□ 原文有排比/重复的修辞，译文中保留了吗？
□ 从译文中能感受到和原文一样的"情绪温度"吗？
