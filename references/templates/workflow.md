# 工作流程

## 🔄 工作流程

```
用户翻译请求
  │
  ├─ ① 语言识别与场景路由
  │     识别源语言和目标语言
  │     匹配S1-S8场景 → 确定默认策略倾向
  │     默认目标语言=中文（除非用户指定）
  │
  ├─ ② 需求澄清（如果关键信息缺失）
  │     目标读者是谁？什么场合用？有格式要求吗？
  │     ⚠️ 只问必要的，快速翻译不追问
  │
  ├─ ③ 查证准备（如果需要）
  │     联网搜索专业术语、文化典故、最新译法
  │     记录搜索过程
  │
  ├─ ④ 五步深译
  │     ├─ 理解原文（字面→深层→语境）
  │     ├─ 确定场景和受众
  │     ├─ 选择翻译策略（直译/意译/本地化）
  │     ├─ 逐层翻译（L1准确→L2通顺→L3地道→L4风格）
  │     └─ 三轮自检（准确性→通顺性→场景适配）
  │
  ├─ ⑤ 本地化适配（如需）
  │     ├─ 文化适配：典故/节日/幽默/颜色符号
  │     ├─ 单位换算：货币/度量衡/日期格式
  │     └─ 称呼转换：敬语/头衔/关系称谓
  │
  └─ ⑥ 输出
        ├─ 译文（直接可用）
        ├─ 翻译说明（策略/决策/本地化处理）
        ├─ 备选译法（如有必要）
        └─ 质量自评（六维分数）
```
